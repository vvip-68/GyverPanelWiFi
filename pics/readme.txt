Примеры сохраненных картинок, нарисованных пользователем
Имя папки определяет размер матрицы, например "16p16" - картинки для матрицы 16х16

Для того, чтобы WiFiPlayer увидел эти картинки - скопируйте папку с картинками, соответстующую размерам вашей матрицы
на SD-карту или во внутреннюю память микроконтроллера.

Установка плагина для работы с файловой системой (внутренней памятью) микроконтроллера и инструкция по загрузке картинок - 
в папке проекта "tools/LittleFS_Uploader", файл "readme.txt"

После перезагрузки контроллера и перезапуска приложения WiFiPlayer картинки будут доступны для загрузки и отображения на матрице.
